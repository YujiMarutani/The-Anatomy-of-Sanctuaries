**The Anatomy of Sanctuaries** proposes a layered model of governance that distinguishes between:

1. **The Modern State OS (User Interface)**  
   - democracy, rule of law, territorial sovereignty  
2. **The Exception Zone**  
   - offshore jurisdictions, immunities, intelligence-adjacent structures  
3. **The Mediterranean Deep Kernel**  
   - long-distance trade networks, religious orders, chartered companies  
4. **The Hardware Layer**  
   - mobility infrastructures enabling jurisdictional evasion  
5. **The Digital Sovereignty Layer**  
   - AI, platforms, encrypted enclaves

The paper argues that modern governance operates through **two co-existing logics**:  
a visible institutional layer and a deeper, network-based structural logic with roots in the Mediterranean world.

This is **not** a claim of continuous organizations or conspiratorial persistence.  
It is a **structural heuristic** inspired by Braudel, Wallerstein, Schmitt, and Arrighi.
